VALORANT

เติมเกม Valorant ด้วย Riot ID หรือ Valorant Points เพื่อใช้ซื้อ Skin ปืน, Battle Pass และไอเทมอื่นๆ ในเกม

เติมเกม Valorant

กรุณากรอก Riot ID ให้ถูกต้อง (เช่น name#TAG)

เลือกจำนวน Valorant Points

เลือกช่องทางการชำระเงิน

วิธีการเติมเกม

  1. กรอก Riot ID ของคุณ (ชื่อ#TAG)
  2. เลือกจำนวน Valorant Points ที่ต้องการ
  3. เลือกช่องทางการชำระเงิน
  4. กดปุ่มยืนยันการเติมเงิน
  5. ทำการชำระเงินตามช่องทางที่เลือก
  6. รอรับ Points ภายใน 1-2 นาที
หมายเหตุ: กรุณาตรวจสอบ Riot ID ให้ถูกต้องก่อนทำการเติมเงิน
